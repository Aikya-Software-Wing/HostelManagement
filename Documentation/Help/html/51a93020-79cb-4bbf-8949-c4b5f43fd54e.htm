Hostel and Mess Management System

Documentation for the Hostel and Mess Management System for RN Shetty Institute of Technology, Bangalore

Admin Credentials

The credentials for the admin account are

  • Username - admin

  • Password - password

Requirements

The requirements are

  • Microsoft SQL Sever 13.0 or above

  • IIS Server 8.0 or above

  • DevExpress v16.1.2 or above

Before First Run

Before the first run, run the T-SQL script found in DatabaseScript folder in HostelManagement project. Log in to the admin account and add hostels and rooms as necessary. After adding hostels and rooms, add users. Please note that only managers can change the fee.
